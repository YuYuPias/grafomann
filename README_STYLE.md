# GRAFOMANN --- STYLE GUIDE

ГрафоМанн --- это не блог и не фотогалерея.

Фотографии являются частью повествования, а верстка --- частью языка
проекта.

## Основные типы страниц

### post

Самостоятельное эссе или рассказ.

### post wide

Эссе, использующее широкие изображения.

### article-intro

Вступительный блок обзорной статьи.

### catalog-item

Самостоятельный объект внутри обзорной статьи.

Используется для: - станции метро; - моста; - памятника; - здания; -
двери; - любого другого объекта.

### item-title

Название объекта.

### item-meta

Краткие сведения: - адрес; - автор; - дата; - стиль; - другие паспортные
данные.

### gallery-note

Пояснение к фотографии или серии фотографий.

## Принцип

Названия модулей должны описывать функцию, а не конкретную публикацию.
