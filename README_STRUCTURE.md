# GRAFOMANN --- PROJECT STRUCTURE

## CSS

-   main.css --- общие стили сайта.
-   articles.css --- структура статей.
-   gallery.css --- визуальный словарь галерей.
-   lightbox.css --- оформление lightbox.
-   sections.css --- страницы разделов.

## JavaScript

-   lightbox.js --- единый просмотрщик изображений.

## Принцип

Каждая новая публикация собирается из существующих модулей. Новые
CSS-классы создаются только тогда, когда появляется новая функция, а не
новая статья.
