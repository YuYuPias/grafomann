# GRAFOMANN

ГрафоМанн --- это проект о внимательном взгляде.

Он посвящен не достопримечательностям, а тем городским объектам, которые
обычно остаются вне поля зрения: дверям, лестницам, павильонам метро,
кладбищам, старым вывескам, промышленной архитектуре, случайным находкам
и забытым деталям.

Фотографии здесь не иллюстрируют текст. Они являются частью
повествования.

Каждая статья строится как последовательность наблюдений, где текст и
изображения обладают равным статусом.

## Документация проекта

-   структуру статей;
-   обзорные публикации;
-   повторно используемые модули;
-   принципы построения страниц.

### README_GALLERY.md

Система фотографий.

-   композиционные принципы.

### README_STRUCTURE.md

-   шаблоны;
-   принципы развития проекта.

------------------------------------------------------------------------

## Основные принципы

-   Новые CSS-классы создаются только тогда, когда появляется новая
    функция.
-   Названия классов описывают назначение, а не конкретную статью.
-   Код должен стремиться к повторному использованию.
-   Форма не существует отдельно от содержания.
-   Простота важнее эффектности.
